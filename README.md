# Numpy Autoencoder

A complete implementation of a Basic Autoencoder from scratch in Python, using only NumPy.

## Project Description
This project implements an autoencoder to compress 32 binary character patterns (5x7 pixels each, defined in `font.h`) into a strict 2D latent space, and then decode them back with a maximum error of 1 incorrect pixel per character.

The neural network features customizable layers, multiple activation functions (Sigmoid, Tanh, ReLU, LeakyReLU, Linear), and different optimizers (SGD, Momentum, Adam) - all implemented entirely from scratch using backpropagation (manual chain rule).

## Dependencies
- Python 3.x
- NumPy
- Matplotlib

Install dependencies using:
```bash
pip install numpy matplotlib
```

## How to Run

A bash script `run_tests.sh` is provided to easily interact with the project.

### Commands

- **Train the autoencoder**:
  ```bash
  ./run_tests.sh train
  ```
- **Generate visualizations** (requires a trained model):
  ```bash
  ./run_tests.sh visualize
  ```
- **Test pixel error for all 32 characters** (requires a trained model):
  ```bash
  ./run_tests.sh test
  ```
- **Preview the font.h characters as ASCII art**:
  ```bash
  ./run_tests.sh preview
  ```
- **Run the full pipeline (Train, Visualize, Test)**:
  ```bash
  ./run_tests.sh all
  ```

## Architecture Details
The default architecture specified in `src/config.py` is:
- **Input Layer**: 35 neurons
- **Encoder Hidden Layers**: 16, 8 neurons
- **Latent Space**: exactly 2 neurons
- **Decoder Hidden Layers**: 8, 16 neurons
- **Output Layer**: 35 neurons

The hidden layers use the `tanh` activation function, while the output layer uses `sigmoid` to match the binary {0, 1} pixel values. Binary Cross-Entropy loss is used along with the Adam optimizer for fast convergence.

## Latent Space & New Characters
The 2D latent space visually separates the characters based on structural similarities. 
A "new character" can be generated by picking a point in the latent space that does not correspond to any training sample. When passed through the decoder, the network maps this novel 2D coordinate back to a 35-dimensional pattern, interpolating the features it learned to form a shape.

## Generated Plots
After running the visualizer, plots will be saved in `outputs/`:
- `outputs/latent_space.png`: Scatter plot of the 32 characters in the 2D latent space.
- `outputs/reconstructions.png`: Grid showing original vs reconstructed characters.
- `outputs/new_character.png`: A decoded novel pattern from an untrained point in the latent space.
